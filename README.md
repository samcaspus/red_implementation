# red_implementation

# Python version >= 3  

# Instalation 
#### Clone command => git clone https://github.com/samcaspus/red_implementation
#### Go to directory => cd red_implementation/Redis/
#### execute command => python redis.py

## Commands Implemented
- Append
- Del
- Exists
- Exit
- Expire
- Get
- Getrange
- Set
- Zadd
- Zrange and Zrange withscores
- Zrankt
